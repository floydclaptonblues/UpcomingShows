BMC Upcoming Shows moving puzzle-collage background add-on.

Upload all files/folders in this ZIP into the root of the UpcomingShows repository.
This package contains optimized background media, puzzle-background.css, puzzle-background.js, and an updated index.html.

Do not overwrite shows.json. It is not included here.

The background tiles use object-fit: contain, so the media is resized for deployment but not cropped.
